Mario Count

Game Rules
You will be given a random target number at the start of the game.

There are four objects below.
#1 Blue Shell
#2 Star
#3 Mushroom
#4 Banana


By clicking on any object, you will have collected a spcific amount of Mario points to your total score.

You win the game by matching your total score to the target number.

You lose the game if your total score goes above the target number.

The value of each object is hidden from you until you click on it.

Each time when the game starts, the game will change the values of each object.

Can you collect the correct items to match the number displayed?

(Press the object you would like to collect and watch your number go up)
